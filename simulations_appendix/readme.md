# Replication Code for Appendix Simulations

This README provides instructions for replicating Figures C1–C13 from the paper's appendix. All figures are generated by modifying settings in `config.R` and running `main.R`.

## Setup

Two methods are available for addressing cross-sectional correlations in panel data:

1.  **Driscoll and Kraay (1998) standard errors** (described in Section B.2.2)
2.  **Two-way clustered standard errors**

The method is controlled by the `cluster_method` parameter in `config.R`:

``` r
cluster_method = "driscoll_kraay"  # Use Driscoll and Kraay (1998) method
cluster_method = "twoway"          # Use two-way clustered method
```

## Figure Generation Instructions

### Figures C1–C3

Set the following parameters in `config.R` and run `main.R`:

``` r
cases = c("simple")
methods = c("fe", "spj", "db", "le")
twoway_cluster = c(FALSE)
```

### Figures C4–C6

These figures use **Driscoll and Kraay (1998) standard errors**.

Set the following parameters in `config.R` and run `main.R`:

``` r
cases = c("correlation")
methods = c("fe", "spj")
twoway_cluster = c(TRUE)
cluster_method = "driscoll_kraay"
```

### Figures C7–C9

These figures use **two-way clustered standard errors**.

Set the following parameters in `config.R` and run `main.R`:

``` r
cases = c("lag")
methods = c("fe", "spj", "dgmm")
twoway_cluster = c(TRUE)
cluster_method = "twoway"
```

### Figure C10

Generating this figure requires an additional step:

1.  Set the following parameters in `config.R`:

``` r
cases = c("dif")
methods = c("fe", "spj")
twoway_cluster = c(FALSE)
```

2.  Run `main.R` to generate the simulation results
3.  Run `04-plot-dif.R` to load the correct plotting function
4.  Rerun the figure generation code in `main.R` to create Figure C10

### Figures C11–C13

Set the following parameters in `config.R` and run `main.R`:

``` r
cases = c("highrho_simple")
methods = c("spj", "fe")
twoway_cluster = c(FALSE)
```
